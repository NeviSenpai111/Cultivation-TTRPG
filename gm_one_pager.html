境隔

Realm Barrier

Defender auto-cancels attacker successes equal to tier gap (max 4 per full realm above).

Bypass methods:

  • Higher-tier technique (−1 realm of barrier per tier above)
  • Dao Comprehension (Novice −1 · Initiate −2 · Adept −3 · Master −4 · Grandmaster full)
  • Heavenly Resentment (1 pt = 1 success, 1/scene)
  • Insight Gambit success (1/fight, retroactive −1)

备会

Session Prep Checklist (30–60 min)
  • Review Seclusion Actions — resolve each PC's slot spending
  • Advance Karma Clocks — pursuers, fated events, timers
  • Plot Session Shape — one quiet, one tension, one action
  • Stat NPCs This Session Needs — 1–2 is typical
  • Reset Running Sheet — countermeasures, pools, tensions
Critical: always note each PC's remaining countermeasures (talismans, forbidden arts, escape items) before session. You must know what they have before declaring killing blows.

悟

In-Combat Insight

Moments of Truth (auto +1 Insight):

  • Survived Above (higher realm opponent)
  • Brink of Death (countermeasure used)
  • Witnessed a Master of your Dao
  • First Blood with a new technique
  • Dao-Resonant kill

Insight Gambit: 1/scene, split dice pool before rolling. Insight dice score only if the moment is meaningful.

Realm modifier: ×1 same · ×2 one above · ×3 two above · ×4 three+ above. None from fighting down.

变通

Ruling Player Invention

Three-question heuristic:

  • Is it mechanically plausible? (If yes, allow it)
  • If not — what existing mechanic is it closest to? Adapt.
  • What does it cost? Qi / Insight / risk — never free.
Default ruling: say yes, set a cost, move on. A slow session from over-adjudicating is worse than a slightly generous call.

Never retcon: no retroactive countermeasures, no "I actually had a talisman." Preparation must have been declared before combat.

家规

House-Rule Decisions · Lock In at Session Zero
  • Insight Gambit works in sparring? (Default: yes, no Brink-of-Death triggers)
  • Resentment refill rate per session? (Default: max 1/session)
  • Multi-Dao comprehension allowed? (Default: yes, Seed's 2× only on one)
  • Karma sharing across party? (Default: individual, not pooled)
  • Witnessing tribulation = Moment of Truth? (Default: yes, 1× per realm level)
  • PC death replacement style? (Options: sect disciple / reincarnation / fresh start)
  • Can PCs found sect branches? (Default: at Nascent Soul+)
  • Core-Bound treasure destruction on wielder death? (Default: yes, shatters)
