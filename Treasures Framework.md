# PART A — TREASURE SYSTEM REFERENCE

## 1. What Is a Treasure

A **treasure** (法宝 / fǎbǎo) is a permanent item — forged, grown, or fallen from heaven — that amplifies a cultivator's capacity beyond their native realm. Treasures are the difference between a cultivator and a walking arsenal. A Core Formation swordsman with a Celestial-tier sword can threaten Nascent Soul opponents; a Nascent Soul cultivator with no treasure is a Nascent Soul cultivator with no treasure.

Treasures are distinct from:

- **Techniques** — techniques are moves the cultivator performs; treasures are objects the cultivator wields or channels through.
- **Pills** — consumable, single-use (see Part C).
- **Formations** — placed, site-bound (see Part C).
- **Mortal tools** — no spiritual nature; useless to serious cultivators past Qi Condensation.

Everything in this document that is permanent, wielded, and spiritual is a treasure.

---

## 2. Anatomy of a Treasure

Every treasure has these fields.

| Field | Purpose |
|---|---|
| **Name** | Flavor and identity. |
| **Tier** | Spirit / Mysterious / Celestial / Void / Nirvana. Sets all base math. |
| **Quality** | Rough / Standard / Refined / Superior / Perfect. Sub-rank within tier. |
| **Category** | Weapon / Defensive / Transport / Utility / Soul / Formation / Puppet / Consumable. |
| **Base Stats** | The dice or value the treasure contributes passively when wielded. |
| **Ability Slots** | Number of distinct named abilities (passive and active). |
| **Binding** | Unbound / Blood-bound / Soul-bound / Core-bound. |
| **Sentience** | Inert / Nascent / Quasi / Fully sentient / Sword Spirit (special). |
| **Dao Affinity** | None, or a Dao the treasure resonates with. |
| **Realm Requirement** | Minimum realm to wield without penalty. |
| **Origin** | Refined / Naturally Formed / Inherited / Living / Celestial Guard. |
| **Evolution Track** | If applicable — how the treasure grows with use. |
| **Tags** | Keywords for system interaction (fire, sword, piercing, sealed, karmic, etc.). |

---

## 3. The Five Tiers

Treasures follow the canonical Xian Ni structure. Tier name = the realm at which the treasure becomes *essential*, not the realm at which it becomes *available*.

| Tier | Canonical Name | Fits Realm | Scarcity in Setting |
|---|---|---|---|
| **Spirit** (灵宝) | Spirit Treasure | Qi Condensation | Common. Sold in sect stores. |
| **Mysterious** (玄宝) | Mysterious Treasure | Foundation Establishment / Core Formation | Uncommon. Clan heirloom tier. |
| **Celestial** (仙宝) | Celestial Treasure | Nascent Soul / Soul Formation | Rare. Named weapons. |
| **Void** (虚宝) | Void Treasure | Soul Transformation / Ascendant | Very rare. Sect-defining artifacts. |
| **Nirvana** (涅宝) | Nirvana Treasure | Nirvana realms | Legendary. Setting-shaping. |

Every tier has a quality sub-rank that multiplies base dice roughly by Rough ×0.75, Standard ×1, Refined ×1.25, Superior ×1.5, Perfect ×2. A Perfect Spirit sword can rival a Rough Mysterious sword on paper — the tier still governs what *abilities* it can host, though, so treasures below your realm hit an ability ceiling no quality can overcome.

## 4. The Eight Treasure Categories

| Category | Typical Forms | Primary Function |
|---|---|---|
| **Weapon** | Swords, spears, bows, fans, whips, bells, hammers | Offense; +dice to attacks, often with special strike abilities. |
| **Defensive** | Armor, cloaks, shielding bracers, jade pendants | Defense; +Realm Barrier dice, damage reduction, or auto-shield reactions. |
| **Transport** | Flying clouds, wind gourds, spirit boats, formation-based vehicles | Movement; flight speed, range, altitude, group transport. |
| **Utility** | Spatial rings, scrying mirrors, beast-taming whips, illumination pearls | Non-combat problem-solving; storage, detection, binding. |
| **Soul** | Soul flags, Nascent Soul pendants, soul-refining lamps | Soul attack/defense; soul nourishment; escape mechanisms. |
| **Formation** | Formation flags, banner sets, array keys | Deploy placed formations from inventory; formation dao amplifiers. |
| **Puppet** | Spirit-iron puppets, puppet cores, Celestial Guards | Autonomous/semi-autonomous combat constructs. |
| **Consumable** | Single-use treasures — sealing talismans, escape stones, heaven-burning beads | One-shot devastating effects; blurs into talismans at low tier. |

**Special case — swords:** Sword-shape Weapon treasures double as Transport via Sword Soaring (see Section 11). This is a category-native capability, not a separate category membership — it doesn't consume ability slots or TP. Flying swords are *weapons that happen to fly,* not Transport treasures. Dedicated Transport treasures (clouds, boats, gourds) exist for non-sword cultivators or for specialized high-speed / group-transport uses.

A treasure can bridge true categories (a fan that is a Weapon and a Utility), but its *primary* category determines its slot economy — a cultivator can effectively wield one Weapon, one Defensive, and a small number of Utility treasures simultaneously. Overburdening causes qi drain and attribute penalties.

## 5. Base Stats & Ability Slots by Tier

This table anchors every treasure's baseline before design decisions.

| Tier | Base Stat (Weapon dice / Defensive dice) | Ability Slots | Qi to Activate (per ability use) | Realm to Wield at Full Power |
|---|---|---|---|---|
| **Spirit** | +2 | 1 | 1 | Qi Condensation |
| **Mysterious** | +4 | 2 | 5 | Foundation Establishment / Core Formation |
| **Celestial** | +6 | 3 | 20 | Nascent Soul / Soul Formation |
| **Void** | +9 | 4 | 75 | Soul Transformation / Ascendant |
| **Nirvana** | +12 | 5+ | 300 | Nirvana Scryer+ |

**Ability slots** hold named effects — passive perks (always on when wielded) or active abilities (spend qi to trigger). A Mysterious sword with two slots might have one passive (wound piercing) and one active (fire burst).

Qi costs align with technique costs at the same tier — this is deliberate so the player-side math stays unified. A Celestial treasure ability costs the same as a Celestial technique (20 qi), but unlike techniques, treasure abilities can be triggered regardless of whether the cultivator knows a matching technique.

## 6. The Binding System

Binding is how a treasure is tied to its wielder. Every treasure exists in one of four binding states; the state determines effectiveness, theft risk, and what happens on death.

### 6.1 The Four Binding States

| State | Effectiveness | How to Bind | Theft / Transfer | On Wielder Death |
|---|---|---|---|---|
| **Unbound** | ~30% — base dice only, no abilities | None. Found/looted. | Anyone can pick up. | — |
| **Blood-bound** | 70% — base dice, 1 ability | Drip of blood + intent; 1 seclusion day. | Can be forcibly seized by a wielder 2+ realms higher. | Unbinds (drops to Unbound). |
| **Soul-bound** | 100% — full abilities | Sliver of soul essence + days of nourishment; 1 seclusion slot. | Cannot be taken from the living. Severs on death. | Unbinds. Treasure rings hollow until rebound. |
| **Core-bound** | 110%+ — special unlocks, treasure grows with wielder | Only at Core Formation+; bind to Golden Core; irreversible. | Utterly impossible to steal. | Treasure enters "dormancy" and rejects all rebinding attempts for years/decades. |

### 6.2 Binding Interactions

- **Wielding above your realm:** even a Soul-bound Celestial treasure in a Foundation Establishment cultivator's hands functions as if Blood-bound — the wielder lacks the soul capacity to power full synchronization. Use the lower effectiveness tier.
- **Wielding below your realm:** no penalty from binding, but base dice contribution is capped by the treasure's tier. A Nascent Soul cultivator wielding a Spirit sword only gets +2 dice.
- **Rebinding an Unbound treasure** requires 1 seclusion slot and a successful Soul roll at target = treasure tier (2/4/7/10/15). Failure wastes the slot and damages the treasure (−1 quality sub-rank, permanent).
- **Breaking a Soul-bound treasure's bond while the wielder lives** requires torture, soul-severing ritual, or the wielder's willing consent. Forcibly broken bonds leave the treasure damaged and the wielder soul-scarred.
- **Core-bound treasures** are canonically the gear of serious cultivators. They cannot be disarmed in the traditional sense — see Section 10 on Realm Gates.

### 6.3 Narrative Stakes

This system deliberately creates a classic cultivation-novel tension: do you kill the enemy cleanly (severing their soul-bound sword into a lootable pickup), or capture them and torture the bond off (faster, but brutal)? Both are canonical. The table's choice reveals the party's Dao.

## 7. Sentience Scaling

Treasures above Mysterious tier are not inert tools. They have gradations of awareness.

| Tier | Sentience Level | Play Effect |
|---|---|---|
| **Spirit / Mysterious** | Inert | Pure tools. No will. |
| **Celestial** | Nascent awareness | Preferences (likes skilled users, resists crude handling). May underperform for an unworthy wielder — GM may quietly reduce dice by 1. |
| **Void** | Quasi-sentient | Has a name, a favored Dao, a personality. Can be "persuaded." Sometimes refuses to attack specific targets (old friends, kin, sworn enemies of its own). |
| **Nirvana** | Fully sentient | Named entity. Has memory, preferences, agenda. May demand specific wielders. A Nirvana treasure in someone's hand is also a relationship. |

**Sentient treasures can:**

- Refuse to activate (drop to lower effectiveness tier mid-combat)
- Demand acts of the wielder (quests, tributes, vows)
- Communicate with a bonded wielder (telepathy, dreams, whispers)
- "Die" if their sentience is shattered (reducing them to permanently Unbound husks)
- Pass across generations, accumulating history that becomes part of their abilities

**GM handling:** sentient treasures are NPCs. A Void-tier sword with a grudge against Demonic cultivators is a character the GM plays, not a stat block. This is where treasure drama lives.

A special category of sentient treasure — the **Sword Spirit** — has its own dedicated rules; see Section 8.

## 8. Sword Spirits

A **Sword Spirit** (剑魂 / jiànhún) is a distinct sentient entity bound to a specific sword — more than the general sentience of Section 7. Sword Spirits are a canonical feature of *Renegade Immortal*: wielders don't just use these swords, they *partner* with them. A sword with a Sword Spirit is functionally a companion that happens to cut.

### 8.1 What Makes a Sword Spirit

A Sword Spirit forms through one of four origins:

- **Soul-infused refining** — the refiner invested a sliver of their own soul, a captured beast spirit, or accepted a dying cultivator's consent to seed the blade with awareness.
- **Resonant awakening** — the sword was wielded with sufficient Sword Dao intensity over centuries that its accumulated qi spontaneously cohered into a mind.
- **Soul absorption** — a cultivator died in the moment of striking, and the sword caught their Nascent Soul. Usually demonic in origin; often unstable.
- **Deliberate gifting** — a Void+ master invested soul energy into a sword over centuries as a conscious act of creation, usually intended as an inheritance.

### 8.2 Tier & Prevalence

Sword Spirits appear only in swords of tier **Celestial and above.** A Mysterious sword can have nascent awareness (standard sentience), but not a true Sword Spirit. The distinction:

- **Sentient treasure (Celestial baseline):** has preferences, can refuse, has a voice.
- **Sword Spirit (Celestial+, optional):** has a *name,* a *history,* its own cultivation, its own techniques, and can act independently.

Not every Celestial sword houses a Sword Spirit — most do not. Those that do are prestige items, typically discovered in ruins, tombs, inheritance, or as deliberate gifts. A party of five should have at most one Sword Spirit among them.

### 8.3 The Sword Spirit Sheet

Every Sword Spirit has its own mini-sheet, played as an NPC:

| Field | Purpose |
|---|---|
| **Name** | The spirit's self-chosen or remembered name. |
| **Realm** | Foundation Establishment through Nirvana Shatterer. Usually 1–2 realms below the current wielder, sometimes above. |
| **Sword Dao Rank** | Always present. Usually higher than the wielder's own Sword Dao. |
| **Techniques** | 1–3 sword techniques the wielder can learn through bonding. |
| **Temperament** | Patient, prideful, wrathful, mournful, curious, loyal. |
| **Preferences / Aversions** | What the spirit wants the wielder to do; what it will refuse. |
| **Origin Tie** | Who the spirit was, what their unfinished business is. |
| **Favor** | 0–10 track, see Section 8.5. |

### 8.4 What a Sword Spirit Does

**In combat:**

- Adds its **Sword Dao rank as bonus dice** to the wielder's sword attacks, capped at (wielder realm + 1) dice so the spirit can't carry an underrealmed wielder infinitely.
- Can **auto-parry** once per combat: the sword defends of its own accord regardless of wielder awareness. This counts as a countermeasure (per the core framework's lethality rules).
- Can **lend a technique**: on the wielder's turn, the wielder "borrows" one of the spirit's known techniques at the spirit's proficiency, once per combat. Qi cost is paid normally from the wielder's pool.

**Out of combat:**

- **Teaches techniques** over seclusion practice. After 3 seclusion slots spent in dedicated practice (and sufficient favor), a technique from the spirit's list becomes the wielder's own permanent technique at Rough proficiency. This is free technique acquisition bypassing normal learning costs — the price is favor and time.
- **Warns of danger**: within the spirit's perception range (a few hundred meters scaling with the spirit's realm), it can sense cultivators, lies, or karmic threats.
- **Accumulates its own karma** — which *can transfer* to the wielder under strain. A Slaughter-hungry spirit that kills through the wielder shares the Black Karma.

**Relationship dynamics:**

- A spirit pushes its wielder toward specific Daos — almost always Sword Dao, sometimes a secondary Dao tied to the spirit's history (Slaughter, Guardianship, Truth, Solitude).
- Spirits develop opinions about wielder choices and communicate them (intrusive thoughts, dreams, pressure on the qi connection).
- A Void+ spirit can, under extreme emotion, **animate the sword to briefly leave its wielder** — a cowardly or oath-breaking wielder may find their sword missing one morning.

### 8.5 The Favor Track

Each Sword Spirit has a **Favor** pool, 0 (hostile) to 10 (bonded beyond question). Favor is the primary narrative lever on the partnership.

**Starting Favor by acquisition method:**

| Acquisition | Starting Favor |
|---|---|
| Forged with wielder's own soul-essence | 7 |
| Earned through significant mortal-world trial or quest | 5 |
| Inheritance from a respected source (master, elder, kin) | 4 |
| Found abandoned in ruins (spirit is grieving/suspicious) | 3 |
| Looted from a defeated enemy (spirit resents rebinding) | 2 |

**Favor rises with (+1 each):**

- A significant victory sharing blood in combat
- Pursuing the spirit's preferred Dao (per seclusion arc)
- Honoring the spirit's history (seeking origin, completing old oaths)
- Feeding (for Slaughter-resonant spirits — kills of specific types)

**Favor falls with (−1 or −2):**

- Using the sword against its kin or preferences (−2)
- Betraying oaths the spirit witnessed (−2)
- Prolonged neglect (−1 per several-month disuse)
- Sharing the sword with a secondary wielder (−1; spirits are jealous)

**Favor thresholds:**

| Favor | State |
|---|---|
| 0 | **Hostile.** Spirit disengages. Treasure functions at Unbound effectiveness. May flee at Void+. |
| 1–3 | **Disaffected.** Bonus dice halved. Refuses to lend techniques. |
| 4–6 | **Working relationship.** Full Section 8.4 benefits available. |
| 7–9 | **Bonded.** +1 bonus die in combat. Spirit may manifest externally once per arc at Nascent Soul+. |
| 10 | **True Bond.** Spirit regards the wielder as their own reincarnation or dearest kin. Unlocks the spirit's signature technique (usually one tier above the treasure's base tier). |

### 8.6 Sword Spirit Cultivation

Sword Spirits cultivate. They gain realm over time, track kills, accumulate Dao insight. Their growth is usually slower than their wielder's — about 1 sub-stage per campaign arc of active use — but they never die of old age, and unlike their wielders they can eventually surpass their original realm of forging.

A Sword Spirit reaching **Nascent Soul** gains the ability to briefly **manifest outside the sword** — a shadowy figure holding the physical blade, usable as a defensive entity once per combat. This is the origin of many legendary "twin-swordsman" stories where one of the two combatants is the sword spirit itself.

### 8.7 Losing a Sword Spirit

Sword Spirits can *die,* which is distinct from the sword being destroyed:

- Sustained heart-demon attack targeting the spirit specifically
- The spirit sacrificing itself to save the wielder (usually a once-per-campaign narrative beat, requires Favor 7+)
- Prolonged wielding by someone whose Dao fundamentally opposes the spirit's (drains over years)
- Being absorbed by a higher-tier Sword Spirit in ritual consumption

A sword whose spirit has died becomes an ordinary sentient treasure of the same tier, but one quality sub-rank lower — a mournful blade that remembers having a voice.

### 8.8 Design Budget Interaction

In Part B's point budget (Section 17), a Sword Spirit **occupies one ability slot but costs zero TP.** It's a structural commitment rather than a budget expenditure — the treasure loses a slot for conventional abilities in exchange for the Spirit's package (bonus dice, technique sharing, auto-parry countermeasure). The Favor track acts as the narrative counterweight.

Tables that find Sword Spirits too strong can optionally require 4 TP and a slot; tables that want them more common can make the slot cost zero as well. Default: one slot, no TP.

### 8.9 Example Sword Spirits

**Solitude's Edge.** The spirit of a Foundation Establishment swordsman who died defending his sister's grave. Refuses to abandon a vulnerable ally. Resents cowardice. Teaches: *Mountain-Guarding Stroke* (defensive). Favored Dao: Guardianship / Solitude.

**The Red Tide.** A Slaughter-forged spirit from a demonic sect's refining. Hungers. Grows with each kill. Whispers for worse prey. Teaches: *Tide of Blood* (AoE). Favored Dao: Slaughter.

**Quiet Stone.** An ancient spirit that has seen six wielders die. Accepts death stoically, expects the same. Teaches no techniques — teaches *presence.* Every round not spent attacking, wielder gains +1 die to their next strike. Favored Dao: Death / Tranquility.

## 9. Dao-Linkage & Resonance

Many treasures resonate with a specific Dao. The resonance affects what they do and who can wield them well.

### 9.1 Resonance Benefits

When the wielder walks the matching Dao:

- **+1 effective tier** when wielded (a Mysterious treasure functions as Celestial)
- Dao-locked abilities activate (sealed effects the non-resonant wielder can't access)
- Evolution potential improves — treasure grows with the wielder (see Section 12)

### 9.2 Mismatch Penalties

Wielder walks an *opposing* Dao:

- **−1 effective tier**
- Treasure may resist activation (sentience check at Celestial+)
- At Void+ tier, opposing-Dao treasures may actively harm the wielder — soul scarring, qi backlash, or refusal to unbind

### 9.3 Common Dao-Linked Types

- **Sword Dao Treasures** — most flying swords; harmonize with Sword Dao. Most common linkage. Sword-Dao walkers are the "correct" wielders of Sword-Spirit-bearing blades (see Section 8).
- **Karma Treasures** — Karma Whip, Karma Bells, Fate-binding Rings. Wang Lin's path.
- **Slaughter Treasures** — Demonic swords, blood-soaked blades, Butcher's Cleavers. Gain power with kills and demand them.
- **Soul Treasures** — Soul Flags, Soul Bells, Soul-Devouring Spears.
- **Space Treasures** — Spatial Rings, Teleportation Stamps, Gate Keys.
- **Time Treasures** — extremely rare; slow/speed/loop effects; often Void+ only.
- **Restriction Treasures** — sealing, binding, formation-based.
- **Elemental Treasures** — fire, water, metal, wood, lightning, ice, wind, earth.

A Slaughter treasure in a healer's hands is useless and possibly hostile. A Karma treasure held by someone with zero karmic weight is inert.

## 10. Realm Gates & Wielder Effectiveness

Canonical hard rule: **Core Formation is the first realm at which a cultivator can wield anything above Spirit-grade at full effectiveness.**

### 10.1 The Effectiveness Table

When a cultivator's realm is below the treasure's recommended realm:

| Realm Gap | Effectiveness | Notes |
|---|---|---|
| At or above recommended | 100% | No penalty. |
| 1 realm below | 70% | −1 ability slot usable; qi cost +50%. |
| 2 realms below | 30% | Base dice only, no abilities; qi cost doubled; qi deviation risk per combat. |
| 3+ realms below | 10% | Dangerous to wield. Treasure may reject or backlash. |

### 10.2 Why This Matters

- A Foundation Establishment cultivator who loots a Celestial sword can't really use it — it's an inventory item until they hit Nascent Soul.
- A Core Formation cultivator with a Mysterious treasure is the *sweet spot* of the system; their treasures carry them hardest here.
- A Nascent Soul cultivator who surfaces with no treasure is weaker than a Core Formation cultivator who has one. This is intentional and canonical.
- The **Core Formation breakthrough** is therefore a massive power spike *not because of the realm alone*, but because the cultivator's entire treasure inventory suddenly activates.

### 10.3 The Dao Workaround

A cultivator whose Dao matches a treasure's linkage treats their effective realm as +1 for wielding purposes. This is how prodigies wield above their station — not by raw strength, but by resonance.

## 11. Sword Soaring — Flying Sword Transportation

**Sword Soaring** (御剑飞行 / yùjiàn fēixíng) is the iconic cultivator transport: a sword-shape Weapon treasure flown by its wielder, who stands upon or beside it and directs it with qi. This is *not* a separate Transport-category ability — it's a **native capability of sword-shape Weapon treasures at Mysterious tier or above**, granted by the category itself. It consumes no ability slot and no TP. Every flying sword you know from *Renegade Immortal* uses these rules.

### 11.1 What Counts as a Sword

Sword Soaring requires sword-shape: katanas, longswords, shortswords, dao/sabres, jian, rapiers, and two-handed greatswords. Blade-beam and whip-blade variants qualify. The following *do not*: hammers, bells, fans, bows, spears (though some legendary spears break the rule with Transport abilities bought on budget), axes.

### 11.2 Requirements to Sword-Soar

- **Sword tier:** Mysterious minimum. Spirit swords can manage brief sword-*hops* (10–20m at a time) but cannot sustain flight.
- **Binding:** Blood-bound or better. Unbound swords cannot be soared.
- **Wielder realm:** Foundation Establishment minimum. Qi Condensation cultivators with a Mysterious+ matched-Dao sword can manage short flights with 50% speed and double qi cost.
- **Qi Control:** no separate check by default — at realm, any cultivator can do it. At lower realms, it's a Qi skill test per flight.

### 11.3 Speed & Range by Sword Tier

Base speed is set by the sword's tier, adjusted by the wielder's Sword Dao rank and load.

| Sword Tier | Base Speed | Altitude | Notes |
|---|---|---|---|
| **Mysterious** | ~60 km/h | Treetop to low hill | Pedestrian for cultivators. |
| **Celestial** | ~300 km/h | Hundreds of meters | Standard for Nascent Soul. |
| **Void** | ~2,000 km/h | Kilometers | Cross-continental in hours. |
| **Nirvana** | Effectively instant regionally | Unlimited | Space-warping at Perfect quality. |

**Modifiers:**

| Condition | Effect |
|---|---|
| Sword Dao Novice | +20% speed |
| Sword Dao Initiate | +50% speed, altitude unrestricted |
| Sword Dao Master | Speed effectively doubled; the "Sword Becomes Self" unlock available (see 11.7) |
| Each passenger (up to 3) | −30% speed, −1 ability slot usable during flight |
| Carrying heavy cargo | −20% speed |
| Storm / suppression field / anti-flight formation | −50% to full grounding |

### 11.4 Qi Cost

Sword Soaring is qi-hungry. Costs scale with tier and travel pace.

| Sword Tier | Sustained Cruise (qi / minute) | Combat-Speed Flight (qi / minute) |
|---|---|---|
| Mysterious | 1 | 5 |
| Celestial | 2 | 10 |
| Void | 4 | 20 |
| Nirvana | 8 | 40 |

Combat-speed flight is the high-dodge, weaving, rapid-climb mode used mid-fight. Most travel is done at cruise.

A Foundation Establishment cultivator's qi pool at standard depth allows roughly 30–45 minutes of cruise on a Mysterious sword before exhaustion. Nascent Soul cultivators on Celestial swords can fly for hours. Peak-realm flyers essentially fly indefinitely on modest quality reserves.

### 11.5 Combat on Swords

Fighting while sword-soaring is possible but tilted.

| Aspect | Modifier |
|---|---|
| **Attack dice** | No penalty (you fight with the sword you ride). |
| **Defense dice** | −2 unless Sword Dao Novice+. Standing on a moving blade is hard. |
| **Mobility** | +2 dice to avoid ground-based attacks. |
| **Ranged-attack vulnerability** | Any ranged attack that successfully wounds forces a Qi or Body check to stay mounted. Failure = dismount. |

**Dismounted in flight = falling.** Without a countermeasure (Escape Talisman, Soul Escape, catching a thrown rope or ally's flying sword, self-cast flight technique, etc.), this is usually lethal at Celestial+ flight altitudes. Countermeasures for falling should be declared *before* combat starts or baked into the wielder's prep.

### 11.6 Sword-Sharing & Group Transport

A single sword can carry its primary wielder plus up to 3 additional passengers, at significant cost to speed and some ability availability. Passengers pay no qi (it all comes from the primary wielder), but:

- Cannot attack from the sword without forcing a control check from the primary wielder (DC varies with the passenger's chaos)
- Cannot use the sword's abilities (only the primary wielder can)
- Dismount if the primary wielder loses control, and fall independently

For sustained multi-person long-distance travel, **dedicated Transport treasures** (spirit clouds, boats, gourds — with ability slots spent on group transport, long-range speed, and so forth) are usually better. Flying swords are personal vehicles.

### 11.7 Sword Becomes Self (Sword Dao Master Unlock)

At **Sword Dao Master,** cultivators unlock the capstone technique of the Sword-Dao transport path: **Sword Becomes Self** (身剑合一 / shēn jiàn hé yī). The cultivator's body and blade merge during flight. This is a Dao-Seed unlock, not a sword ability; it applies to any Mysterious+ sword.

Effects:

- Flight speed upgraded by one tier equivalent (a Celestial sword flies at Void baseline)
- Qi cost halved (body-as-sword is massively more efficient)
- No passenger capacity — the body *is* the sword
- The cultivator cannot be individually targeted during flight; they count as a single combat object (the sword-form). Ranged attacks hit or miss the blur; no dismount check.
- Can pass through gaps the blade can fit through (arrow slots, pinhole formations, etc.)

**Drawback:** during the merge, the cultivator cannot use techniques that are not sword techniques. Their own body is locked into the sword-form for the duration.

This is the canonical "sword-light streaking across the sky" image from cultivation fiction. It's rare, expensive to earn, and unforgettable.

### 11.8 Flight Interactions with Sword Spirits

A sword with a Sword Spirit (Section 8) can *self-pilot.* If the wielder is unconscious, dying, or dismounted, the sword's spirit can carry the wielder's body away at reduced speed (50% base) without requiring the wielder to channel qi. This is a once-per-arc rescue, and burns 1 point of Favor.

High-Favor spirits sometimes exhibit flight quirks — a proud spirit prefers high altitudes, a mournful spirit prefers specific routes, a protective spirit refuses to fly the wielder into known danger without warning them three times.

### 11.9 Non-Sword Transport

For treasures that aren't sword-shape, Transport functionality must be bought on the ability budget (see Section 18). A cloud, gourd, boat, or carpet that flies does so because TP was spent to make it fly — with the advantage of typically better group transport capacity, altitude, and weather resistance than a sword of the same tier.

The design tension is real: a Celestial sword gets flight *and* combat abilities on its budget; a Celestial cloud must spend several TP on flight, leaving less for everything else. This is why sword cultivators are canonically overrepresented.

---

## 12. Evolution — How Treasures Grow

Treasures above Mysterious tier can evolve. Evolution is the mechanism by which a treasure becomes *more* than what it was refined to be.

### 12.1 Evolution Triggers

A treasure evolves when it accumulates a critical mass of one or more of:

- **Kills** (for Weapons — blood absorbed, karma accumulated)
- **Damage Taken** (for Defensive — impact resonance)
- **Distance Traveled** (for Transport — including sword-soared swords)
- **Deeds Performed** (for Utility and Soul items)
- **Dao Exposure** (any treasure used consistently within a resonant Dao's practice)

### 12.2 Evolution Effects

Each evolution tick (GM-tracked; roughly one per major campaign arc of heavy use) does one of:

- Adds one ability slot (up to the tier's cap)
- Upgrades quality by one sub-rank
- Gains a Dao affinity if it didn't have one
- Develops nascent sentience (if already at that tier's threshold)
- Unlocks a named ability keyed to the event (a sword that killed a Nascent Soul may gain "Soul-Bite")
- For Sword-Spirit treasures: advances the spirit's own cultivation by one sub-stage

### 12.3 Crossing Tiers

A Perfect-quality treasure at one tier, after sufficient evolution, can cross the threshold into the next tier — becoming a Rough-quality treasure of the higher tier. This is how legendary weapons are actually made in the setting: not solely through forging, but through *use*.

A sword forged as Mysterious, wielded by three Nascent Souls across four centuries, may retire as a Rough Celestial blade — and at Celestial, it may *develop a Sword Spirit* if no one ever seeded it deliberately. This is the treasure side of the cultivation logic: everything can advance if it endures.

---

## 13. Special Types

### 13.1 Naturally Formed Treasures

Not refined by any cultivator. Condensed from heaven and earth over millennia. Spirit stones themselves are the commonest example; Dao Fruits, Heavenly Materials, and certain geographic anomalies (a mountain that is a Void-tier Restriction treasure) also qualify.

Naturally formed treasures are often stronger than refined ones at the same tier — they have no crafter's limits — but are usually unbound, fickle, and tied to their location. Moving one often degrades it.

### 13.2 Inherited Treasures

Passed down through sects, clans, families. These always come with conditions:

- **Karma attached** — the former wielder's unfulfilled oaths, blood debts, or enemies may become yours
- **Recognition requirements** — bloodline, oath, sigil, or sect insignia needed to fully bind
- **Sentient preferences** — the treasure may remember its last master and compare you unfavorably

Inherited treasures are the best plot hooks in the system. Every such treasure is a tether to its history. Inherited Sword-Spirit swords are especially potent — the spirit remembers the previous wielder and has expectations.

### 13.3 Living Treasures (Beast Treasures)

Refined from the essence, heart, bone, or soul of a spirit beast. Occupy a gray zone between treasure and creature — they may retain the beast's instincts, appetites, and hatreds.

A flying sword forged from a dragon's spine may demand heights. A cloak woven from a demon-wolf's pelt may howl during full moons. Living treasures are always Celestial+ tier and always sentient to some degree.

### 13.4 Celestial Guards (Puppet-Treasures)

Canonical *Xian Ni* puppet type: refined from cultivators (canonically willing — consent is a hard requirement) whose body is hardened, soul amplified, organs withered, and divine sense wiped. The result is a sentient-but-unfeeling combat construct.

Celestial Guards use their own sub-ranking:

| Rank | Combat Power Equivalent |
|---|---|
| **Copper** | Foundation Establishment Peak |
| **Silver** | Core Formation Peak |
| **Gold** | Nascent Soul / Soul Formation (can challenge Nirvana Scryers) |
| **Platinum** (GM extension) | Soul Transformation+ |

Celestial Guards:

- Obey their master absolutely
- Cannot cultivate further but do not age
- Are Void-tier treasures in value
- Occupy the master's Puppet treasure slot
- Creating one requires willing sacrifice (orthodox) or compulsion (demonic — a dark subsystem for tables that want it)

### 13.5 Spatial Rings

Canonically ubiquitous — essentially every Foundation Establishment+ cultivator has one. A Spatial Ring contains a pocket dimension for storage.

| Ring Tier | Storage Volume |
|---|---|
| **Spirit** | Small room (~30 m³) |
| **Mysterious** | Large room (~200 m³) |
| **Celestial** | Small building (~2000 m³) |
| **Void** | Manor / estate (~10,000 m³) |
| **Nirvana** | Small world |

**Restrictions:**

- Living creatures normally cannot be stored (exceptions at Void+)
- Soul-bound rings cannot be looted from the living
- Can be stolen from the dead (unbound on death as usual)
- Pocket dimensions large enough may be *entered* — a Void-tier ring is essentially a portable cave-realm and a common narrative device

Spatial Rings occupy a Utility slot but are so standard the system treats them as infrastructure rather than equipment choice.

## 14. Economy

### 14.1 Spirit Stone Values (low-grade spirit stones)

| Tier | Price Range | Quality Multipliers |
|---|---|---|
| **Spirit** | 10 – 200 | Refined ×2, Superior ×4, Perfect ×10 |
| **Mysterious** | 200 – 5,000 | same |
| **Celestial** | 5,000 – 100,000 | same |
| **Void** | 100,000 – 5,000,000 | priceless at Perfect |
| **Nirvana** | Myth-items — not reliably traded | — |

**Sword Spirit premium:** a Celestial+ sword with a genuine Sword Spirit typically commands 2–5× the base tier price, and most Sword-Spirit swords are *not for sale at any price* because the spirit itself refuses to be rebound by commerce.

### 14.2 Theft & Loot

Unbound treasures are fair game. Blood-bound treasures become unbound when the wielder dies. Soul-bound treasures become unbound at death. Core-bound treasures enter dormancy — you can take them, but they won't work for anyone for a long while.

This creates three distinct loot behaviors at the table:

- **Clean kills** produce unbound treasures (if the wielder was blood- or soul-bound)
- **Capture and torture** can break soul-bonds faster but degrades the party's karma
- **Sparing the wielder** may gain favor but denies the loot

This is a moral pressure point the system creates on purpose.

### 14.3 Inheritance

Sect treasures, clan heirlooms, and master-disciple gifts bypass the spirit-stone economy but come with the karmic strings in Section 13.2. An inherited treasure is never *just* a weapon.

# PART B — TREASURE CREATION FRAMEWORK

## 15. Design Principles

Five rules for treasure design — the same shape as the technique framework's rules, tuned for items.

1. **A treasure is a character, not a stat block.** Design its voice first. "A patient sword that has waited sixty years for a worthy hand" will suggest mechanics better than a spreadsheet.
2. **One clear combat role, one clear narrative hook.** Treasures that are "good at everything" are boring and break budget math.
3. **At least one ability must change with Dao, binding, or evolution.** A treasure that never grows is a commodity. Treasures in *Xian Ni* are relationships.
4. **Restrictions are features.** A blade that only unsheathes in moonlight is more interesting than one that unsheathes anywhere. Restrictions also refund TP (see Section 20).
5. **Respect the realm gate.** A Celestial treasure should make a Nascent Soul cultivator *significantly* stronger. If it doesn't, you've under-designed.

---

## 16. The 8-Step Design Process

1. **Concept.** One sentence: "What is this treasure?" ("A bone flute that calls ghosts.")
2. **Tier.** Spirit → Nirvana. Sets TP budget and ability slot count.
3. **Category.** Weapon / Defensive / Transport / Utility / Soul / Formation / Puppet / Consumable. (Note: sword-shape Weapons get Sword Soaring free.)
4. **Base Stats.** Apply tier baseline (see Section 5).
5. **Spend TP.** Use the budget on abilities from the Effect Cost Menu (Section 18). If the treasure has a Sword Spirit, reserve one slot for it (no TP cost, see Section 19).
6. **Apply Sentience & Dao-Linkage.** At Celestial+, this is mandatory. Decide whether this includes a Sword Spirit.
7. **Restrictions.** Add thematic limits for TP refunds (Section 20).
8. **Evolution Hook.** Define one trigger and its effect (Section 21).

---

## 17. The Treasure Point (TP) Budget

TP mirrors the technique framework's PP values for cross-math compatibility.

| Tier | TP Budget | Ability Slots | Notes |
|---|---|---|---|
| **Spirit** | 4 | 1 | Workhorse gear. No sentience. |
| **Mysterious** | 8 | 2 | Standard cultivator gear. Sword-shape Weapons get Sword Soaring free. |
| **Celestial** | 16 | 3 | Must have Dao affinity OR nascent sentience. Sword Spirit is an option here. |
| **Void** | 32 | 4 | Must have both Dao affinity AND quasi-sentience. |
| **Nirvana** | 64 | 5+ | Fully sentient. GM NPC. Setting-level. |

Quality sub-rank modifies TP: Rough −20%, Standard 0, Refined +25%, Superior +50%, Perfect +100%. A Perfect Spirit has 8 TP but still only 1 ability slot — sub-rank makes existing effects stronger, not more numerous.

A Sword Spirit, when included, **occupies one ability slot but costs zero TP** (see Section 19.3).

| **Reflective shield** | 4 (active) | Returns half damage of next incoming attack. |

## 19. Sentience, Sword Spirits, & Dao-Linkage Design

### 19.1 Sentience (mandatory at Celestial+)

When designing a Celestial+ treasure, choose sentience flavor:

- **Temperament** — patient / prideful / wrathful / mournful / curious / loyal
- **Preference** — what it likes in a wielder (courage, cruelty, learning, restraint)
- **Aversion** — what it refuses to do (kill its own kind, obey cowards, strike specific targets)
- **Voice** — how it communicates (whispers, dreams, chimes, pressures, visions)

Sentience does not cost TP. It's mandatory and structural; use it to make the treasure interesting rather than to gate power.

### 19.2 Dao-Linkage (mandatory at Celestial+)

Pick one Dao. Specify:

- **Benefit** (+1 effective tier when wielder walks matching Dao)
- **Locked ability** — one of the treasure's slotted abilities only functions for matching-Dao wielders
- **Penalty** — opposing-Dao wielders take −1 tier and risk rejection

Dao-linkage does not cost TP either — it's a structural feature of higher-tier treasures.

### 19.3 Sword Spirits (optional at Celestial+; only for sword-shape Weapons)

A Sword Spirit is a full NPC bound to the treasure (see Section 8). Adding one to a sword treasure:

- **Occupies one ability slot** (out of the tier's 3/4/5+ slots)
- **Costs zero TP** — the spirit is not an ability but a structural partner
- **Requires a full Spirit sheet** — name, realm (1–2 below wielder), Sword Dao rank (always present, usually Initiate+), 1–3 techniques, temperament, preferences/aversions, origin tie, and starting Favor (Section 8.5)
- **Replaces nascent sentience** — the sword doesn't also need the general Section 7 sentience on top; the spirit is the sentience

A Sword Spirit is roughly +4 to +6 TP of effective power for its slot. The reason it's a slot rather than TP is that the Favor system provides the narrative counterweight — a Spirit's benefits evaporate if the wielder loses the relationship. This is a very different design pressure than a fixed-cost ability.

**Design guidance for Sword Spirits:**

- Give them a clear preference that the wielder *might* conflict with. Alignment alone is boring.
- Give them one technique that will actually get used in the campaign, not three that stay on the sheet.
- Give them a past worth discovering. A Sword Spirit with no origin hook is wasted design space.
- Start Favor low enough that earning it is meaningful but high enough that the partnership actually functions at acquisition. The 2–7 range is appropriate.

---

## 20. Restrictions (Budget Refunds)

A treasure with meaningful restrictions refunds TP. Stack freely. Each restriction refunds as noted.

| Restriction | Refund | Notes |
|---|---|---|
| **Condition-locked** | +2 TP | Only works under moonlight, during specific emotion, in specific locations, etc. |
| **Oath-bound** | +3 TP | Requires a Heart Oath to wield; breaks the treasure if oath broken. |
| **Hunger** | +2 TP | Requires feeding (blood, karma, souls, qi) periodically or becomes inert. |
| **Single-use per arc** | +4 TP | One killer ability usable once per campaign arc, not combat. |
| **Dao-exclusive** | +3 TP | Only works for one very specific Dao; useless otherwise. |
| **Rejection risk** | +2 TP | Every combat, 1-in-6 chance the treasure refuses to activate. |
| **Karmic cost** | +2 TP | Each use adds Black Karma. |
| **Lifespan drain** | +3 TP | Each activation burns weeks/months of cultivator lifespan. |
| **Sentient refusal (strong)** | +3 TP | Treasure actively refuses specific actions. |

**Note on Sword Spirits:** the Favor track is already a built-in restriction on Sword-Spirit treasures and should not be additionally taxed. Do not double-count — adding "Sentient refusal" on a Spirit-bearing sword doesn't refund TP, because the Favor system covers that ground.

Restrictions are thematic anchors. A wraithblade that hungers for lives is more characterful *and* more powerful-within-budget than an equivalent sword with no restriction.

---

## 21. Evolution Design

Every Mysterious+ treasure should have at least one evolution hook. Define:

- **Trigger** — the specific condition (kills of specific kind, damage taken, years of continuous wear, being blooded in specific events, Favor threshold for Spirit swords)
- **Threshold** — roughly, amount needed (e.g., "100 cultivator kills" or "survives three Tribulations" or "Favor reaches 10")
- **Effect** — what unlocks (new ability slot, quality bump, tier promotion at extreme thresholds, Dao affinity, for Spirit swords: the spirit's signature technique)

A treasure without evolution is a commodity; a treasure with evolution is a narrative engine.

## 22. Four Worked Examples

### 22.1 Ashen Whisper (Spirit, Weapon)

**Concept.** A fire-scorched short sword that hisses when unsheathed near a lie.

**Tier:** Spirit. **Budget:** 4 TP. **Slots:** 1.
**Category:** Weapon. **Quality:** Standard.
**Base stats:** +2 dice to Body/Qi attacks.

**Budget spend:**

- Elemental (fire): 2 TP (passive)
- Detection (lies within 5m): 2 TP (passive) — combined into single ability slot as "The Whisper"

**Total: 4 / 4.**

**Final:**

- Ability 1 (passive): *The Whisper* — adds fire damage to attacks; hisses audibly when anyone within 5m speaks a direct lie.
- Binding: Blood-bound (typical for Spirit).
- No sentience, no Dao-linkage (below tier).
- Sword Soaring: N/A (Spirit tier can only manage sword-hops, not sustained flight).
- Evolution hook: *after 20 lies exposed, the blade gains a second passive — +1 die vs. targets who have lied to the wielder within the last day.*

### 22.2 Still-Lake Mirror (Mysterious, Defensive / Utility)

**Concept.** A palm-sized water-jade mirror that turns the cultivator's reflection into their shield.

**Tier:** Mysterious. **Budget:** 8 TP. **Slots:** 2.
**Category:** Defensive / Utility hybrid. **Quality:** Refined (+25% — 10 TP).
**Base stats:** +4 defensive dice.

**Budget spend:**

- Reflective shield (active): 4 TP — once per combat, returns half damage from next incoming attack
- Scrying (short-range): 3 TP (active) — see places the wielder has been
- Restriction: Only works where water is visible or within reach: +2 TP refund
- Restriction: Rejection risk (sentience-adjacent): +2 TP refund

**Total: 7 TP of effects, 4 TP in refunds, for 11 TP effective spend against 10 budget. Close enough (off by 1 — quality sub-rank's 25% is rough).**

**Final:**

- Ability 1 (active): *Reflected Blow* — once per combat, return half damage.
- Ability 2 (active): *Still Scrying* — view a known location; requires water.
- Restrictions: inoperative far from water; sometimes sulks.
- Binding: Soul-bound.
- Evolution hook: *after surviving 3 combats where it saves the wielder, unlocks a third passive ability — +2 dice to all Soul defense rolls while held.*

### 22.3 The Karma-Threading Bell (Celestial, Soul)

**Concept.** A bone-and-bronze bell that rings only in the presence of the wielder's enemies of fate.

**Tier:** Celestial. **Budget:** 16 TP. **Slots:** 3.
**Category:** Soul. **Quality:** Standard.
**Sentience:** Nascent — mournful, patient; prefers wielders with unfulfilled vows. Refuses to ring for false enemies.
**Dao-Linkage:** Karma Dao. +1 effective tier for Karma-walkers.
**Base stats:** +6 Soul attack dice.

**Budget spend:**

- Soul damage (active): 4 TP — +4 extra dice to Soul attacks; bypasses Realm Barrier by 1 realm
- Karma manipulation (active): 4 TP — reveal a target's Black Karma weight; visible to all witnesses (Face damage)
- Detection (passive): 2 TP — rings faintly when fated enemy within 1km
- Oath binding (active, Dao-locked): 6 TP — force a Heart Oath if the target has active karmic debt to the wielder
- Restriction: Oath-bound — wielder must have at least one unfulfilled Heart Oath to use the bell at all: +3 TP refund
- Restriction: Karmic cost — each use adds 1 Black Karma: +2 TP refund

**Total: 16 TP of effects, 5 TP in refunds, for 11 TP net spend against 16 budget. Clean.**

**Final:**

- Ability 1 (passive): *Thread Sense* — detect fated enemies within 1km.
- Ability 2 (active): *Ring of Judgment* — Soul attack, pierce 1 realm of Barrier, reveal target's Black Karma to all witnesses.
- Ability 3 (active, Karma-locked): *Bind the Debt* — force a Heart Oath on a karmically indebted target.
- Binding: Soul-bound, strongly prefers Core-bound at Nascent Soul.
- Sentience: refuses to ring for non-fated targets; mourns when unused too long.
- Evolution hook: *after binding three Heart Oaths in anger, the bell gains a fourth ability slot — a single-use detonation that annihilates one oath-breaker. This consumes the bell.*

### 22.4 Chill-Moon Sabre (Celestial, Weapon + Sword Spirit)

*The canonical Sword-Spirit flying-sword example. This is the treasure that best demonstrates the integrated Sword-Soaring and Sword-Spirit systems in play.*

**Concept.** An inherited flying sabre, forged on a moonlit peak by a sword-Dao hermit who died cultivating alone. The sabre carries the hermit's spirit, and the hermit has preferences.

**Tier:** Celestial. **Budget:** 16 TP. **Slots:** 3 (one occupied by the Sword Spirit, two for TP abilities).
**Category:** Weapon (sabre) — performs Sword Soaring natively (Section 11).
**Quality:** Standard.
**Base stats:** +6 dice to sword attacks.

**Sentience:** Via Sword Spirit — mournful, patient, prizes restraint; aversion to killing the unarmed or sleeping.
**Dao-Linkage:** Sword Dao. +1 effective tier for Sword-Dao wielders.

**Sword Spirit** (slot 3, 0 TP): ***Jiao the Hermit***
- Realm: Foundation Establishment Peak
- Sword Dao: Initiate
- Techniques: *Mountain-Guarding Stroke* (defensive sword technique, +3 defense dice for one round, 5 qi) and *Moon-Culling Sweep* (Celestial-tier offensive sweep, usable once per day through Jiao)
- Temperament: Mournful, patient
- Preferences: Restraint, respect for the dying, dedication to Sword Dao
- Aversions: Killing the unarmed, sleeping, or children
- Origin: Died alone on a peak during his own breakthrough tribulation; never told his sister. His unfinished karma is reaching her descendants.
- Starting Favor: 5 (inherited from a respected senior)

**Budget spend (slots 1 and 2, 16 TP):**

- Ability 1 (passive): *Frost Edge* — adds cold elemental damage, slows opponents 1 tier below wielder. 4 TP.
- Ability 2 (active): *Moon-Cutter Strike* — AoE cleave in a line (3 TP), pierces 1 realm of Barrier (2 TP), +4 damage dice on this strike (4 TP), wounds dealt count as one tier higher (3 TP). 12 TP. Activated at 20 qi.

**Total: 16 / 16.** No restrictions needed — the Favor track and Jiao's preferences are the narrative counterweight.

**Sword Soaring** (native to sword-shape Celestial Weapon):

- Base speed: ~300 km/h, hundreds of meters altitude
- +50% with wielder at Sword Dao Initiate (Jiao's Favor at 6+ effectively teaches the wielder Sword Dao over time, so this is achievable)
- Qi cost: 2 qi/minute cruise, 10 qi/minute combat-flight
- Can carry 1 passenger at reduced speed
- If wielder is unconscious, Jiao can self-pilot to safety (1 Favor point, once per arc)

**Binding:** Soul-bound; prefers Core-bound at Nascent Soul.

**Evolution hook:** Once Jiao's Favor reaches 8 *and* the wielder has completed Jiao's unfinished business (made contact with the hermit's descendants and delivered the truth of his death), the sabre gains a fourth ability slot and unlocks Jiao's personal signature technique — ***Shattering Winter*** (Celestial+ tier, winter storm across battlefield, 40 qi, usable once per day).

**Play note:** at True Bond (Favor 10), Jiao can briefly manifest externally once per combat as a shadowy hermit figure wielding the sabre, acting as a Foundation Establishment Peak combatant for one round. This is a once-per-arc rescue-scale effect.

## 24. Pills (Alchemy Outputs)

Pills are consumable — one dose, one effect. The alchemy subgame produces them.

### 24.1 Pill Tiers

| Tier | Typical Effects | Quality Sub-Rank Effect |

## 25. Formations (Placed Arrays)

Formations are inscribed patterns that create persistent effects in a location. They're site-bound, slow to deploy, and massively force-multiplicative.

### 25.1 Formation Tiers

| Tier | Scale | Typical Effect |
|---|---|---|
| **Spirit** | Room | Minor ward, concealment, qi-gathering |
| **Mysterious** | Building / small courtyard | Significant defense, cultivation acceleration, deception |
| **Celestial** | Estate / mountain | Sect protection, tribulation diversion, beast-barrier |
| **Void** | Valley / city | Region-defining, space-warping |
| **Nirvana** | Realm-scale | Setting-altering — worlds hidden, paths sealed, time dilated |

### 25.2 Formation Design (Simplified Budget)

Formations are more complex than pills — multiple linked effects.

| Tier | TP Budget | Flag Count | Deploy Time |
|---|---|---|---|
| **Spirit** | 4 | 3 flags | 1 day |
| **Mysterious** | 8 | 6 flags | 1 week |
| **Celestial** | 16 | 9 flags | 1 month |
| **Void** | 32 | 36 flags | 6 months |
| **Nirvana** | 64 | 81 flags | Years |

### 25.3 Common Formation Effects

| Effect | Cost |
|---|---|
| Concealment (from detection) | 2 |
| Physical barrier (1 realm's worth) | 3 |
| Qi-gathering (+25% to cultivators within) | 4 |
| Illusion (small) | 3 |
| Illusion (landscape-level) | 8 |
| Entrapment (victim cannot leave) | 6 |
| Tribulation diversion (1 wave) | 6 |
| Space-warping (enter elsewhere) | 8 |
| Time-dilation (2× inside) | 12 |
| Soul-wounding for intruders | 4 |

### 25.4 Fuel

Formations require fuel to persist:

- **Spirit stones** — continuous; costs per week depend on tier
- **Spirit veins** — passive geological fuel; setting-dependent
- **Sacrifice** — blood, qi, souls; demonic
- **One-time charge** — formation runs until discharged, then collapses (common for tribulation diversion)

### 25.5 Quick Formation Examples

- **Cloaking Veil Array (Spirit, 3 flags):** Concealment (2) + qi muffling (2). 4 TP. 1-week fuel per low-grade spirit stone.
- **Nine-Dragon Sealing Formation (Void, 9 flags):** Entrapment (6) + physical barrier (6) + soul-wounding (4) + illusion-landscape (8) + qi-drain-intruders (4) + tribulation diversion (6). Uses 9 primary flags + 72 secondary, 32 TP. Fuel: one Void-tier spirit stone per month, or one cultivator's lifespan per decade (demonic variant).

## 26. Talismans (Single-Use Inscribed Paper)

Talismans are Formation Dao products in portable form — a paper or silk strip inscribed with a simplified formation pattern, charged, and discharged in one use.

### 26.1 Talisman Tiers

Talismans target roughly one tier *below* the formation they emulate — portability costs power.

| Tier | Cost to Craft | Effect Scale |
|---|---|---|
| **Spirit** | Cheap; mass-produced | Single small effect |
| **Mysterious** | Few hours per talisman | One meaningful combat effect |
| **Celestial** | Days per talisman | Countermeasure-tier |
| **Void** | Weeks; rare | Saves a life |
| **Nirvana** | Usually plot items | Reverse a disaster |

### 26.2 Design (Simplest Budget)

Each talisman is a single-effect item. Its TP cost equals the effect's cost on the Formation menu (Section 25.3), but the talisman's tier is the formation's tier minus one (minimum Spirit).

**Concrete:** a "reduce tribulation by 1 wave" effect is 6 TP on the Formation menu, so it produces a **Celestial** talisman (because the effect is Void-scale when placed). Talismans are always one tier below their effect's native scale — this is why countermeasure talismans are always pricey.

### 26.3 Common Talisman Types

| Talisman | Tier | Effect | Typical Use |
|---|---|---|---|
| **Escape Talisman** | Spirit | Short teleport (100m) | Breaking out of ambush |
| **Shield Talisman** | Spirit/Mysterious | Block one attack of tier ≤ wielder's | Common disciple gear |
| **Soul-Escape Talisman** | Celestial | Triggers Soul Escape countermeasure for non-Nascent Soul | Life-saver |
| **Tribulation Diversion Talisman** | Celestial | Removes 1 wave | Breakthrough insurance |
| **Binding Talisman** | Mysterious | Paralyze target of tier ≤ wielder | Combat control |
| **Explosive Talisman** | Spirit–Void | Tier-appropriate blast | Siege / assassination |
| **Fall-Catching Talisman** | Spirit | Softens a fall to non-lethal | Specifically designed for sword-soaring dismounts |

### 26.4 Mass Production

A crafter can produce talismans in batches during seclusion — unlike formations, the same pattern can be stamped repeatedly. A Formation Dao crafter at Initiate can produce roughly one Spirit-tier talisman per day of focused work; Mysterious costs a week; Celestial costs a month.

This is why a party with a Formation specialist suddenly has a drawer full of single-use options the rest of the table draws from.

## 27. Cross-Craft Interactions

- **Alchemy + Artifact Refining:** Pills can be sealed *into* treasures as charges — a Celestial sword that contains five doses of a healing pill releases one per strike. Costs ability slots.
- **Formation + Artifact Refining:** A treasure can house a formation as one of its abilities (the Deployable Formation entries in Section 18.5). Effectively a one-shot reserve array in the wielder's pocket.
- **Talismans as Counter-Talismans:** Higher-tier talismans can nullify lower-tier formation effects, and vice versa. A Celestial talisman can briefly collapse a Mysterious array. Anti-flight talismans can ground a sword-soaring opponent for seconds — enough for a dismount.
- **Pills + Techniques:** Some pills are designed to be consumed during specific technique activations, boosting a technique by one tier for that use. These are Mysterious+ pills and always come with severe side effects.
- **Sword Spirits + Techniques:** A Sword-Spirit sword that teaches a technique effectively adds that technique to the wielder's permanent repertoire after dedicated seclusion. This bypasses the usual technique-acquisition economy and is one of the strongest reasons to pursue a Sword-Spirit inheritance.

These interactions make crafter-specialized parties exponentially stronger than generalist ones. This is canonical *and* intended — the game rewards specialization across the party, not within a single character.
